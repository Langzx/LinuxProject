### 实现的功能:
1. 实时进度条，拖动进度条控制视频播放进度
2. 暂停/恢复视频
### 未实现的功能:

1. 进度条过渡不太流程
2. 进度条在拖动减少进度时，会出现视频与进度条不同步，在原地等待视频
3. 线程之间的优化，中断信号ctrl+c会死机

1. 视频的缩放会导致视频速度减慢
2. 视频的颜色深度有问题导致只有黑白加错位
3. mplayer从有名管道得到指令后，本应该回复在有名管道内，但是确回复在标准输出中，导致只能使用无名管道接收信息，导致设计思路转变
